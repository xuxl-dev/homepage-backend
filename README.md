
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

# About modules
| module name | description |
|---|---|
| announcement  |  面向某一group的通知  |
| auth  | 鉴权  |
| chat | 聊天 |
| chatgroup | 聊天组 |
| <del>comment</del> | <del>评论区，已弃用</del>  |
|db|数据库|
|group|用户组|
|internal-message|站内信|
|notification|通知(QQBot, WeChatBot, mailing, internal-message)|
|offline-message|离线消息|
|task-flow|任务流|
|thunder|雷消息（类似DING）|
|user|用户|
|usermeta|用户元数据(目前主要维护在线情况)|
